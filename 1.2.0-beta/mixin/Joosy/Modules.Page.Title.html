Method Details

~ (void) title(title, separator = ' / ')

Note: Title will be reverted on unload.

Sets the page HTML title.

Examples:

class TestPage extends Joosy.Page
  @title 'Test title'

class TestPage extends Joosy.Page
  @title -> I18n.t('titles.test')

Parameters:

  • title (String) — Title to set.
  • separator (String) — The string to use to .join when title is an array
