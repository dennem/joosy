Mixin: Joosy.Modules.Resources.IdentityMap Concern

Defined in: source/joosy/modules/resources/identity_map.coffee

Overview

The basic implementation of Identity Map allowing you to automatically make all the instances of the same model to be a single JS Object.

Examples:

class Model extends Joosy.Resources.Hash
   @concern Joosy.Modules.Resources.Model
   @concern Joosy.Modules.Resources.IdentityMap

foo = Test.build id: 1
bar = Test.build id: 1   # bar === foo
