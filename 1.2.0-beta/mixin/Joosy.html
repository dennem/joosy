~ (void) buildUrl(url, params)

Basic URI builder. Joins base path with params hash

Examples:

Basic usage 

Joosy.buildUrl 'http://joosy.ws/#!/test', {foo: 'bar'}    # http://joosy.ws/?foo=bar#!/test

Parameters:

  • url (String) — Base url
  • params (Hash) — Parameters to join
