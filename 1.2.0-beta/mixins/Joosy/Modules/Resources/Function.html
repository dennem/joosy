    Mixin: Joosy.Modules.Resources.Function

    Overview

    Module allowing to emulate Fn-based instances in JS

    Examples:

    class Foo extends Joosy.Module

  @extend Joosy.Module.Function

  constructor: (value)
    @value = value

__call: -> @value

foo = Foo.build 'test'
typeof(foo)             # 'function'
foo()                   # 'test'
