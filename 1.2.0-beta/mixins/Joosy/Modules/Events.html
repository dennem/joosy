    Method Summary

    • ? (void) eventsNamespace(actions) Creates events namespace

    ? (void) eventsNamespace(actions)

    Creates events namespace

    Examples:

    namespace = @entity.eventsNamespace, ->
  @bind 'action1', ->
  @bind 'action2', ->

namespace.unbind()

    ? (void) wait(name, events, callback)

    Waits for the list of given events to happen at least once. Then runs callback.

    Parameters:

    • events (String|Array) — List of events to wait for separated by space
    • callback (Function) — Action to run when all events were triggered at least once
    • options (Hash) — Options

    ? (void) unwait(target)

    Removes waiter action

    Parameters:

    • target (Function) — Name of waiter to unbind

    ? (void) bind(name, events, callback)

    Binds action to run each time any of given event was triggered

    Parameters:

    • events (String|Array) — List of events separated by space
    • callback (Function) — Action to run on trigger
    • options (Hash) — Options

    ? (void) unbind(target)

    Unbinds action from runing on trigger

    Parameters:

    • target (Function) — Name of bind to unbind

    ? (void) trigger(event, data...)

    Triggers event for {bind} and {wait}

    Parameters:

    • Name (String) — of event to trigger

    ? (void) synchronize(block)

    Runs set of callbacks finializing with result callback

    Examples:

    Basic usage

    • block (Function) — Configuration block (see example)
