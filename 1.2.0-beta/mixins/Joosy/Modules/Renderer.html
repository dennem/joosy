    Overview

    Core DOM rendering mechanics

    Method Summary

    • ? (void) included() Defines class-level helpers: @view and @helpers View (@view): Sets the curent template by specifying its name or lambda Helpers (@helpers): Lists set of helpers' namespaces to include
    • ? (void) render(template, locals = {}, parentStackPointer = false) Renders given template with given locals
    • ? (void) renderDynamic(template, locals = {}, parentStackPointer = false) Dynamically renders given template with given locals Whenever any of assigned locals triggers changed event, DOM will automatically be refreshed
    • ? (void) __assignHelpers() Converts all possible @helper arguments to the objects available for merge
    • ? (void) __instantiateHelpers() Collects and merges all requested helpers including global scope to one cached object
    • ? (void) __instantiateRenderers(parentStackPointer) Defines local @render* methods with proper stack pointer set
    • ? (void) __render(dynamic, template, locals = {}, parentStackPointer = false) Actual rendering implementation
    • ? (void) __renderingStackElement(parent = null) Template for the rendering stack node
    • ? (void) __renderingStackChildFor(parentPointer) Creates new rendering stack node using given pointer as the parent
    • ? (void) __removeMetamorphs(stackPointer = false) Disables and unbinds all dynamic bindings for the whole rendering stack
