Class Method Details

. (void) scroll(element, options = {})

Note: If you use animated scroll joosy will atempt to temporarily fix the height of your document while scrolling to prevent jump effect.

Sets the position where page will be scrolled to after load.

Examples:

class TestPage extends Joosy.Page
  @scroll '#header', speed: 300, margin: -100

Parameters:

  • element (jQuery) — Element to scroll to
  • options (Hash)

Options Hash: (options):

  • speed (Integer) — Sets the animation duration (500 is default)
  • margin (Integer) — Defines the margin from element position. Can be negative.
