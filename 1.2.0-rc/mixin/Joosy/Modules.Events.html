~ (void) wait(events, callback)
~ (void) wait(name, events, callback)

Waits for the list of given events to happen at least once. Then runs callback.

Parameters:

  • name (String) — Custom name for the binding
  • events (String) — List of events to wait for separated by space
  • events (Array) — List of events to wait in the form of Array
  • callback (Function) — Action to run when all events were triggered at least once
  • options (Hash) — Options

Returns:

  • (String) — An ID (or custom name) of binding

Overloads:

~ (void) wait(events, callback)

Uses internal unique ID as the name of the binding

~ (void) wait(name, events, callback)

Allows to pass custom name for the binding
