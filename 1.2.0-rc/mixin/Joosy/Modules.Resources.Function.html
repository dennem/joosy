Mixin: Joosy.Modules.Resources.Function Concern

Defined in: source/joosy/modules/resources/function.coffee

Overview

Module allowing to emulate Fn-based instances in JS

Examples:

class Foo extends Joosy.Modules.Resources.Hash
  @concern Joosy.Module.Function

foo = Foo.build foo: 'bar'
typeof(foo)                  # 'function'
foo('foo')                   # 'bar'
