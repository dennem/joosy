. (void) find(where, options = {}, callback = false)

Requests the required resource from backend

Parameters:

  • where (String) — id or an array of ids for interpolated path
  • options (Hash) — Path modification options
  • callback (Function) — (error, instance, data) -> ...

Options Hash: (options):

  • action (String) — Adds the given string as a last path element i.e. /resources/trololo
  • url (String) — Sets url for request instead of generated i.e. /some/custom/url
  • params (Hash) — Passes the given params to the query

# (void) collectionPath(ids = [], options = {})

Builds collection path

Examples:

Basic usage 

Resource.collectionPath() # /resources/

Nested resources 

Resource.collectionPath(['admin', Resource.build 1]) # /admin/resources/1/resources

Parameters:

  • ids (Array) — Interpolation arguments for nested objects
  • options (Object)

Options Hash: (options):

  • url (String) — Manually set URL
  • action (String) — Action to add to the URL as a suffix
