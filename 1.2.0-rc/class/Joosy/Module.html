. (Object) merge(destination, source, unsafe = true, deep = false)

Simple and fast shallow merge implementation.

This is here due to: https://github.com/andrewplummer/Sugar/issues/100. This bug was closed and we got some performance but this implementation is still like 10x fater for basic tasks.

Parameters:

  • destination (Object) — Object to extend
  • source (Object) — Source of new properties
  • unsafe (Boolean) — Determines if we should rewrite destination properties
  • deep (Boolean) — Whether merge should go down recursively into nested objects

Returns:

  • (Object) — The new and mighty destination Object
