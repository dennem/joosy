    + (void) find(where, options = {}, callback = false)

    Requests the required resources from backend

    Parameters:

    • where (String) — Possible values: 'all', id. 'all' will query for collection from collectionPath. Everything else will be considered as an id string and will make resource query for single instance from memberPath.
    • options (Hash) — Path modification options
    • callback (Function) — Resulting callback (will receive retrieved Collection/Resource)

    Options Hash: (option):

    • parent (Joosy.Resources.REST) — Sets the given resource as a base path i.e. /parents/1/resources
    • parent (String) — Sets the given staring as a base path i.e. /trololo/resources
    • from (String) — Adds the given string as a last path element i.e. /resources/trololo
    • params (Hash) — Passes the given params to the query
