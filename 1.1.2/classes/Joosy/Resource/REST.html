    + (void) find(where, options = {}, callback = false)

    Requests the required resources from backend

    'all' will query for collection from collectionPath. Everything else will be considered as an id string and will make resource query for single instance from memberPath. (will receive retrieved Collection/Resource)

    Parameters:

    • where (String) — Possible values: 'all', id.
    • options (Hash) — Path modification options
    • callback (Function) — Resulting callback

    Options Hash: (options):

    • parent (Joosy.Resource.REST) — Sets the given resource as a base path i.e. /parents/1/resources
    • parent (String) — Sets the given staring as a base path i.e. /trololo/resources
    • from (String) — Adds the given string as a last path element i.e. /resources/trololo
    • params (Hash) — Passes the given params to the query
